# Privacy Policy — Prompt Enhancer Pro

Last updated: March 2026

Prompt Enhancer Pro does not collect, store, or transmit any personally
identifiable information.

The extension sends selected text only to webhook URLs that the user
personally configures. No data is sent to the extension developer.

All configuration data is stored locally on the user's device using
Chrome's chrome.storage.local API and is never synced to external servers.

The extension does not track browsing history, user activity, or any
other personal data.
